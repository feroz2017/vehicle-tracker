# Vehicle tracker frontend

Vehicle tracker client-app and frontend.

Minimal React setup with React and Vite.

What is included:
- [Vite](https://vite.dev/) build tool.
- [Tailwind CSS](https://tailwindcss.com/) CSS framework.
- [Typescript](https://www.typescriptlang.org/)
- [ESLint](https://eslint.org/) code linter
- [Prettier](https://prettier.io/) code formatter

## How to run

Install dependencies:
```bash
npm install
```

Run development build:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Run code linter with ESLint:
```bash
npm run lint
```

Run code formatter with Prettier:
```bash
npm run format
```

## Environment variables

Set backend API URL in `.env`.

See [.env.development](./.env.development) for example `.env` configuration.
